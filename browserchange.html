68. Is there any need to change the web browsers to support HTML5

No need to change the web browsers to support htmls

❗ When You Might Need to Update or Change Your Browser You only might need to change browsers if:
You’re using an old version of Internet Explorer (especially IE8 or below).
Your browser hasn't been updated in years and doesn’t support newer HTML5 elements or APIs (like local storage, etc.).


Modern Browsers Already Support HTML5 HTML5 has been around since 2014, and all major browsers have supported it for years. These include: Google Chrome
Mozilla Firefox
Microsoft Edge
Safari
Opera
If you're using any reasonably recent version of these, you’re good to go.
